CubeSphereTexture - a Qt3D Earth Rendering experiment

You'll need to download tile files for satellite imagery and elevation using
the python scripts provided in the scripts subfolder.

Once downloaded, don't forget to update the paths to you textures in
mapzenimagerytileprovider.cp

There are 2 rendering options, edit the main.cpp file to load the proper QML
file for the rendering option you selected.
